# Halleyx Dashboard Frontend

## Overview
Custom dashboard and customer order management frontend built with React.

## Features
- Dashboard widgets
- Orders CRUD
- Date filters
- Responsive layout

## Tech Stack
- React
- React Router
- Recharts

## Run Locally
npm install
npm start

## Build
npm run build
